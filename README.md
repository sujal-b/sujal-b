## 🔬 Current Focus — DGE-RAG

> *Authoring a research paper on a novel **Spherical Vector Quantizer** that transforms 384-dimensional dense float arrays into discrete **Coarse & Fine Codes** — reducing memory footprint by orders of magnitude while maintaining retrieval precision.*

| Area | Detail |
|:---|:---|
| 🧠 **Innovation** | Discrete quantization of dense embedding spaces for scalable RAG retrieval |
| 📐 **Architecture** | Spherical codebook design with hierarchical coarse-to-fine decoding |
| ☁️ **Infrastructure** | Azure-hosted pipeline with PyTorch quantization backend |
| 🏷️ **Keywords** | `Quantization` `Latent Space Optimization` `MLOps` `RAG` |
